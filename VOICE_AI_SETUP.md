# تعليمات تثبيت تحسينات الإدخال الصوتي

## الخطوة 1: إضافة ملف التحسينات

تم إنشاء ملف `voice-ai-enhancement.js` في المجلد الرئيسي للمشروع. هذا الملف يحتوي على:
- دالة `handleAiActionEnhanced` المحسنة مع جميع أوامر الذكاء الاصطناعي الجديدة
- تحديث تلقائي لواجهة الإدخال الصوتي لاستخدام الدالة المحسنة

## الخطوة 2: تحميل الملف في الصفحة

أضف السطر التالي في ملف HTML الرئيسي (مثل dashboard.html أو index.html) بعد تحميل app.js:

```html
<script src="voice-ai-enhancement.js"></script>
```

## الخطوة 3: الأوامر الصوتية المدعومة

بعد التثبيت، يمكنك استخدام الأوامر الصوتية التالية:

### الأوامر الأساسية:
- "أنشئ عقد صيانة" - فتح نموذج إنشاء عقد صيانة
- "إنشاء عرض سعر" - فتح نموذج إنشاء عرض سعر
- "إنشاء زيارة" - فتح نموذج إنشاء زيارة كشفية
- "نقل الزيارة [رقم] إلى [اسم الفني]" - نقل زيارة إلى فني محدد
- "إعادة توزيع الزيارات" - إعادة توزيع الزيارات بناءً على التحليل الجغرافي

### أوامر الذكاء الاصطناعي:
- "توصيات" أو "تقرير" - إنشاء تقرير التوصيات الذكي
- "تحسين عرض" - تحسين عرض سعر بناءً على بيانات الموردين
- "حلل تقرير" - تحليل تقرير زيارة وتوليد عرض سعر تلقائي
- "ملف فني" أو "ملف مهني" - إنشاء الملف المهني للفني

### أوامر التنقل:
- "اذهب إلى العقود" أو "افتح العقود" - الانتقال إلى صفحة العقود
- "انتقل للزيارات" أو "عرض الزيارات" - الانتقال إلى صفحة الزيارات
- "افتح البلاغات" - الانتقال إلى صفحة البلاغات
- "عرض عروض الأسعار" - الانتقال إلى صفحة عروض الأسعار
- "اذهب للمخزون" أو "عرض قطع الغيار" - الانتقال إلى صفحة المخزون
- "افتح الفنيين" أو "عرض الفريق" - الانتقال إلى صفحة الفريق
- "انتقل للتقارير" - الانتقال إلى صفحة التقارير
- "افتح الإدارة الذكية" أو "الذكاء الاصطناعي" - الانتقال إلى صفحة الإدارة الذكية

### أوامر البحث:
- "ابحث عن [كلمة]" - البحث في البيانات
- "بحث [كلمة]" - البحث في البيانات

### أوامر العرض:
- "حالة النظام" أو "إحصائيات" أو "نظرة عامة" - عرض النظرة العامة والإحصائيات

### أوامر الاعتماد:
- "اعتماد تقرير" - اعتماد تقرير محدد

### أوامر المراجعة:
- "مراجعة الزيارات" - الانتقال لمراجعة الزيارات
- "مراجعة العقود" - الانتقال لمراجعة العقود

## الخطوة 4: الاختبار

1. افتح الصفحة الرئيسية للتطبيق
2. اضغط على زر الإدخال الصوتي (🎙️)
3. جرب أحد الأوامر المذكورة أعلاه
4. تأكد من تنفيذ الأمر بشكل صحيح

## ملاحظات مهمة:

- جميع الأوامر تتطلب صلاحيات المدير (owner, company_admin, admin)
- بعض الأوامر قد تطلب إدخال بيانات إضافية عبر prompt
- النظام سيطلب تأكيد قبل تنفيذ العمليات الحساسة
- الإجابات الصوتية ستكون باللغة العربية
- التعرف الصوتي المحسن يوفر دقة أفضل مع تنظيف النص تلقائياً
- يتم عرض النص المتوسط أثناء الاستماع لتحسين التغذية الراجعة

## استكشاف الأخطاء:

إذا لم يعمل الإدخال الصوتي:
1. تأكد من تحميل ملف voice-ai-enhancement.js بعد app.js
2. تحقق من دعم المتصفح لـ Web Speech API
3. افتح console المتصفح للتحقق من الأخطاء
